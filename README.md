### Description

"The Wikipedia of Travel", this is a full-stack MERN application where users can find a collection of "bucket-list" travel destinations, read reviews, and even contribute their own.

---

**_Technologies Used_**

---

- Mongo
- Express
- React.js
- Node.js
- JavaScript
